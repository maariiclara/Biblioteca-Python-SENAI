# 🐍 Biblioteca Virtual SENAI - Sistema em Python

Projeto acadêmico desenvolvido durante o curso técnico no **SENAI**, com o objetivo de criar um sistema de gerenciamento para uma biblioteca virtual utilizando Python.

O sistema permite visualizar livros, realizar empréstimos, organizar gêneros e gerenciar informações da biblioteca através do terminal.

---

## ✨ Sobre o Projeto

A proposta do projeto foi desenvolver um sistema funcional de biblioteca virtual utilizando lógica de programação e estruturas de dados em Python.

O sistema possui funcionalidades voltadas para usuários e bibliotecários, permitindo controle de empréstimos, visualização de livros e gerenciamento do catálogo.

---

## 👩‍💻 Desenvolvimento

Todo o sistema foi desenvolvido individualmente por mim, incluindo:

- Estrutura completa do sistema
- Organização da lógica em Python
- Sistema de empréstimos
- Cadastro de livros
- Controle de disponibilidade
- Separação por gêneros
- Área administrativa da bibliotecária

---

## 📌 Funcionalidades

- 📚 Listagem de livros disponíveis
- 🔎 Organização por gêneros
- 📖 Sistema de empréstimos
- 👤 Registro de aluno e sala
- ✅ Controle de disponibilidade dos livros
- ➕ Cadastro de novos livros
- 👩‍🏫 Área exclusiva da bibliotecária
- 📋 Visualização de empréstimos realizados
